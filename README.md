# 🎧 Audio Classification Web App

This project is a web-based audio classification system that uses an ONNX deep learning model to classify audio inputs in real-time.

---

## 🚀 Features

- 🎤 Audio input processing
- 🤖 ONNX model inference in browser
- ⚡ Fast and lightweight
- 🌐 Simple UI using HTML & JavaScript
